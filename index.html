Web Development Focus

This site is built for ITMD 361 using semantic HTML and external CSS. It demonstrates clean structure, validation compliance, readable layout, and disciplined organization across multiple pages.

What This Project Demonstrates

  • Three separate pages with consistent navigation
  • Semantic structure using header, main, section, and footer
  • Relative links to local media files stored in a directory
  • Accessibility basics including descriptive alt text
  • W3C validation for each page

What I Am Building This Semester

My goal is to develop a strong technical base that supports more advanced work later in the course. I am focusing on clean structure first, then progressively adding complexity as new topics are introduced.

  • Writing HTML that is readable, structured, and validated
  • Using GitHub with meaningful commits that reflect progress
  • Organizing files using a clear directory structure
  • Communicating ideas clearly using headings and sections
  • Improving accessibility habits early instead of retrofitting later

Media Requirement
