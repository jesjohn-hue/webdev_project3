Web Development Focus

This site is built for ITMD 361 using semantic HTML, external CSS, and JavaScript. It demonstrates clean structure, validation compliance, responsive layout, and interactive features across multiple pages.

What This Project Demonstrates

  • Three separate pages with consistent navigation
  • Semantic structure using header, main, section, and footer
  • Responsive design using media queries
  • JavaScript interactivity with a dynamic image slider
  • Google Maps API integration with custom features
  • W3C validation for HTML and CSS
